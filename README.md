# Installationsanleitung

Dieses Projekt ist eine interaktive 3D-Simulation im Browser.

## Wichtiger Hinweis
Da das Projekt moderne JavaScript-Module verwendet, kann die `index.html` **nicht einfach per Doppelklick geöffnet werden** (Browser-Sicherheitsrichtlinien blockieren dies). Die Datei muss über einen lokalen Server ausgeführt werden.

## Möglichkeit 1: Visual Studio Code (Empfohlen)
Am einfachsten funktioniert es mit der "Live Server" Erweiterung in VS Code:

1. Öffnen Sie den Projektordner in **VS Code**.
2. Installieren Sie (falls nötig) die Erweiterung **Live Server** (von Ritwick Dey).
3. Machen Sie einen **Rechtsklick** auf die Datei `index.html`.
4. Wählen Sie **"Open with Live Server"**.
5. Das Projekt öffnet sich automatisch im Browser.

## Voraussetzungen
- **Internetverbindung**: Das Projekt lädt die 3D-Bibliothek (Three.js) direkt aus dem Internet. Bitte stellen Sie sicher, dass Sie online sind.
